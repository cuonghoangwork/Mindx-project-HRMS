# HR Management System Website

A lightweight HR management dashboard built with HTML, CSS, and JavaScript.

## Features

- Dashboard with HR stats
- Employee directory with search
- Add employee modal form
- Leave request listing
- Payroll overview cards
- Local storage persistence for employee data

## Run

1. Open `index.html` directly in your browser, or
2. Use VS Code/Cursor Live Server extension and run the page.

## Project Structure

- `index.html` - UI structure
- `styles.css` - styling and responsive layout
- `app.js` - data and interactive logic
